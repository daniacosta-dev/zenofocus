## [1.0.0] - 2026-03-06

### Added
- ⏱ Pomodoro timer with focus, short break, and long break modes
- 🎯 Visual progress ring with countdown
- 📋 Task management — create, edit, delete tasks with categories and estimated pomodoros
- ✅ Task completion with confirmation modal
- 📅 Agenda with visual timeline (7am–8pm) and scheduled tasks
- 🔔 Automatic reminders 10 minutes before scheduled tasks
- 📊 Stats — daily score, weekly history, time by category, completed tasks
- 🪟 Floating widget — always-on-top mini timer with drag support
- 🖥 System tray icon with show/hide and widget toggle
- 🔔 Native GNOME notifications via `notify-send`
- ⚙️ Settings panel — standard (25/5/15) or custom timer durations
- 🔁 Configurable autostart for breaks and focus sessions
- 🔢 Configurable pomodoros before long break
- 🔊 Sound on session end (Web Audio API)
- 🌍 Full bilingual support — Spanish and English
- 💾 Persistent storage — tasks and settings saved to disk automatically
- 📖 Built-in Pomodoro method guide in settings
- ⚠️ Mode change warning modal when timer is running
